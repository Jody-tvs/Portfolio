Présentation du projet

"Folie d'Armes" est un site e-commerce fictif que j'ai conçu pour démontrer mes compétences en développement web full stack, alliant créativité et fonctionnalité. Ce projet original est conçu pour proposer une expérience utilisateur moderne et fluide.

Technologies utilisées :

  • HTML5 / SASS : Structure et design responsive.
  • JavaScript : Interactivité et dynamique du site.
  • React.js : Interface utilisateur rapide et réactive.
  • Node.js : Côté serveur pour gérer les requêtes.
  • MySQL : Base de données pour la gestion des produits et commandes.
  • Modèle MVC : Architecture claire et scalable.

Fonctionnalités :

  • Design mobile-first : Optimisation pour toutes les tailles d'écran.
  • Système de panier et paiement sécurisé : Processus d'achat fluide.
  • Formulaire de contact fonctionnel : Pour poser des questions ou faire des suggestions.
  • Dashboard d’administration complet : Gestion des produits, commandes et utilisateurs.
